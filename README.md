# Prueba N°3 Frontend con Angular 18 y Boostrap 5

Proyecto generado con [Angular CLI](https://github.com/angular/angular-cli) versión 18.2.21.

Estilos manejados puramente con [Bootstrap](https://getbootstrap.com/) versión 5.3.8.


## Ejecutar Localmente

Clonar el proyecto

```bash
  git clone https://github.com/ketudev/PruebaAngular
```

Dirigirse al directorio

```bash
  cd PruebaAngular
```

Instalar dependencias

```bash
  npm install
```

Levantar el servidor

```bash
  ng serve
```

Abrir en navegador

```bash
  http://localhost:4200/
```
